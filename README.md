# Gemini Side Panel

Brave Side Panel extension that integrates Gemini with advanced productivity features.

## Version 3.5
- **Icon**: `icon.jpg` is used for extension branding.
- **Store Asset**: `catching.jpg` is included for promotional use in the web store.
- **Features**: 
  - Alt+S: Toggle Side Panel
  - Alt+Q: Explain selection (Forces new Gemini chat)
  - Smart Toolbar: Summarize, YKS-style, Code Analysis, and Deep Evaluation buttons.
  - Robust Injection: Resilient against page loading delays and Brave Shields.
  - No-Reload Architecture: Stable browsing without infinite refresh loops.
